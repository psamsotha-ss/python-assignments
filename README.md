# Smoothstack Python Assignments

### Set up virtual environment

```shell
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $
```

Install Python modules

```shell
(venv) $ pip install -r requirements.txt
```

### Run a program

```shell
(venv) $ python day5/bmi.py
over normal under 
```

### Deactivate virtual environment

```shell
(venv) $ deactivate
$
```
